# Synthetic network demo

This example runs the Phase 2 public-safe urban drainage monitoring demo.

It creates a synthetic multi-sensor network with:

- one rainfall gauge
- two level sensors
- one flow sensor
- battery/status telemetry
- rainfall-event detection
- sensor-response summaries
- status-rule summaries
- rainfall/level/flow plot
- synthetic dashboard map
- HTML report

Run from the repository root:

```bash
python examples/network_demo/run_network_demo.py
```

or:

```bash
urban-drainage-qaqc network-demo
```
